Aclaraciones
------------

En la carpeta sim se encuentra el código ejecutable que corre el simulador, el cual está escrito en Python.
Cabe destacar que el programa tiene como dependencias algunas librerías de python que se asume están instaladas como wxPython, matplotlib y numpy.
Se presenta el informe en formato pdf, y en la carpeta doc, los archivos utilizados para generarlo, utilizando latex, por si desea modificarlo. 
A su vez, se puede encontrar las diapositivas de la presentación final y el video del simulador.
